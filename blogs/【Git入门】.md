# 常用命令

```shell
git init 
git add .
git commit -m "first commit"
git remote add origin https://github.com/Snailclimb/test.git
git push -u origin "master"

git branch -M main
git push -u origin "main"

git reset --hard HEAD^
git reset --hard HEAD~3

清楚目前提交区：git checkout .
拉取最新：git pull
```
